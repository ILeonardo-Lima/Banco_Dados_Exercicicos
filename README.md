## Sobre o curso
Curso técnico de **Programador Back-end** oferecido pelo SENAI, com foco em desenvolvimento de sistemas, lógica de programação e especialização em tecnologias back-end, com forte ênfase em:

- Modelagem de dados
- SQL puro 
- Consultas

## Tecnologias utilizadas nestes exercícios

- MySQL
- PostgreSQL
- DBeaver / MySQL Workbench
- SQL puro
